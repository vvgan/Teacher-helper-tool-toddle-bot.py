# Teacher-helper-tool-toddle-bot
Help teachers bulk up load students' (Multiple Grade) works to the Toddle Website.   
这是一个为教师开发的自动化工具，旨在帮助快速批量上传学生的作业到 Toddle 平台。

### 🚀 功能特点
- 支持多年级学生作品批量处理。
- 自动化操作，节省教师办公时间。
